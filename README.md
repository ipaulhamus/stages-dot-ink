## Stages.Ink: A Splatoon Schedule Desktop Widget ##
<details>
  <summary>Features</summary>

  * Displays the modes, stages, and times for the current rotations in Splatoon 3 as a desktop widget
  * Can be moved anywhere on the desktop and has sizing options
  * Includes multiple different color themes to select, with planned support for custom themes in the future
  * Has the option to automically open the widget upon device startup
</details>
<details>
  <summary>How To Install / Use</summary>

 (This application is currently only available for Windows, I plan on making releases for Linux and Mac soon, but there are some issues to iron out)
 
  1. Download the .exe installer for the application from the "Releases" page
  2. Execute the installer, it should just be a one-click install
  3. Once you've opened the app, select the "Settings" icon on the bottom left corner to configure it to your liking
  4. The "Settings" page is scrollable, this includes options for the X/Y position of the window, colors, and auto-start
  5. Once you've set up the app, you're done!
  
</details>
